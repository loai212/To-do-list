

To-Do List Web App

    Developed a user-friendly web application for task management.
    Technologies Used: Node.js, Express, PostgreSQL, HTML, CSS.
    Implemented features such as task categorization, due dates, and reminders.
    Designed a RESTful API to handle CRUD (Create, Read, Update, Delete) operations for tasks.
    Focused on responsive design to ensure usability across various devices.
